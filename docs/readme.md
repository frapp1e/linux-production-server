# Linux Production Server

Proyecto práctico de administración de sistemas Linux en entorno productivo.

El objetivo es simular la gestión completa de un servidor corporativo siendo el único administrador, aplicando medidas de seguridad, monitorización, automatización y resolución de incidencias reales.

## Tecnologías
- Rocky Linux 9.7
- OpenSSH
- firewalld
- SELinux
- Fail2ban
- Apache / Nginx (pendiente)
- Prometheus / Grafana (pendiente)
